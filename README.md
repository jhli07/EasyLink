# EasyLink
一个简约的端对端文件传输工具。

一、项目简介
EasyLink是一个开源的网页端文件传输工具。它允许用户在不同设备之间直接传输文件，无需安装任何软件，只需打开浏览器即可使用。

二、核心特点

1. 隐私优先：文件直接在设备间传输。
2. 零安装：纯网页应用，打开即用，支持所有现代浏览器。
3. 跨平台：支持手机、电脑、Windows、Mac、Linux、iOS和Android。
4. 快速连接：通过5位数字代码快速建立连接，需要在同一网络下连接。

三、使用方法

1. 手机端：打开页面，生成设备代码。
2. 电脑端：打开页面，生成设备代码。
3. 输入任意一方的设备代码进行连接。
4. 传输文件：拖拽文件到页面，点击发送，接收方自动接收。

四、适用场景

· 会议室共享资料
· 教室分发作业
· 手机与电脑互传文件
· 私密文件传输

五、温馨提示

· 推荐传输小于100MB的文件
· 确保双方网络连接稳定
· 传输过程中请保持页面打开
· 于同一局域网内传输

总结：EasyLink是一个隐私、简单、跨平台的网页版文件传输工具，适合中小文件的快速共享。
